# Project Rules

1. The app prepares a student for a 20-minute oral exam in "Outils d'analyse du Web".
2. The question bank must be based only on the course notebooks.
3. Do not introduce topics absent from the notebooks.
4. The app must be terminal-based first.
5. The app will eventually support multiple choice, true/false, fill-in-the-blank, predict-output, short-answer, and oral-explanation questions.
6. Wrong questions must be repeated later.
7. Progress must eventually be saved in progress.json.
8. Add tests for core logic whenever implementing features.
9. Before generating real questions, consult docs/course_content_map.md.
